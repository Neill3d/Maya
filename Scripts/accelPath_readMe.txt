	FBIK_AccelPath
  Сценарий для расчета траектории ускорения при перемещении объекта. Для каждого выделенного объекта создается коробка с именем "accel_" + <название объекта>, которой назначается итог вычисления траектории для данного объекта.
  Подобную полученную информацию можно использовать для анализа качества движения объекта, т.е. выявлять лишние изломы, ускорения, замедления, различного рода скачки.
